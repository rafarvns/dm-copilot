# Templates de Personas

## Template Padrão

```
# Persona: [Nome fictício]

**Perfil**
- Idade: 
- Ocupação: 
- Nível tech: (leigo / intermediário / avançado)

**Objetivos**
- O que quer alcançar usando o produto?

**Frustrações (Dores)**
- O que o irrita no processo atual?

**Comportamentos**
- Como resolve o problema hoje (workarounds)?
- Com que frequência usaria o produto?

**Frase que define essa persona**
> "..."
```

## Exemplo Preenchido

```
# Persona: Carla, Recepcionista de Clínica

**Perfil**
- Idade: 34 anos
- Ocupação: Recepcionista há 6 anos
- Nível tech: Intermediário (usa sistemas de agendamento básicos)

**Objetivos**
- Agendar consultas rapidamente sem erros de conflito de horário
- Avisar pacientes de forma automática sobre confirmação e lembretes

**Frustrações (Dores)**
- Planilha de Excel usada hoje gera conflitos de horário frequentes
- Precisa ligar manualmente para confirmar todas as consultas do dia seguinte

**Comportamentos**
- Usa WhatsApp para confirmar consultas manualmente
- Imprime a agenda do dia todo manhã

**Frase que define essa persona**
> "Eu perco 1 hora por dia só confirmando consulta que podia ser automático."
```

## Dica: Anti-persona
Também documente quem **não** é seu usuário, para evitar escopo creep:
- Quem não vai usar o produto?
- Que problema você **não** está resolvendo?
